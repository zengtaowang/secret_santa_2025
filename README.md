# 🎅 Secret Santa 抽签

麻辣格林伍德 Secret Santa 抽签应用 - 2025年12月7日火锅局

一个简单的网页应用，用于随机分配神秘圣诞老人配对。

## 🌐 在线使用

直接访问：**https://zengtaowang.github.io/secret_santa/**

## ✨ 特性

- 🎲 随机配对生成（确保没有人抽到自己）
- 📱 响应式设计，支持手机和电脑
- 💾 浏览器本地存储（无需服务器）
- 📊 实时抽签进度追踪
- 🔒 管理员面板（可查看所有配对）
- 🎨 精美的渐变色设计

## 🎯 使用方法

### 参与者操作：
1. 打开链接：https://zengtaowang.github.io/secret_santa/
2. 从下拉菜单选择你的名字
3. 点击"查看我的对象"按钮
4. **保守秘密！** 不要让别人看到你的屏幕

### 管理员操作：
1. 在底部输入管理员密码
2. 点击"管理员登录"
3. 可以查看所有配对关系和进度
4. 可以重新生成配对或重置进度

**默认管理员密码**: `admin123`

## ⚙️ 技术说明

- 纯 HTML/CSS/JavaScript 实现
- 使用 Bootstrap 5 样式框架
- 数据存储在浏览器的 localStorage
- 无需后端服务器
- 可直接在 GitHub Pages 上托管
